// SNMP Setup for windows 
1. Install the SNMP Feature (GUI Method)
	Open Settings > Apps > Optional features.
	Click View features next to "Add an optional feature".
	Search for "SNMP", check both Simple Network Management Protocol (SNMP) and WMI SNMP Provider, then click Next and Install. 
	
2. Configure the SNMP Service
	Press Win + R, type services.msc, and press Enter.
	Find and double-click SNMP Service.
	On the General tab, set Startup type to Automatic and click Start.
	i>Go to the Security tab.
		a> Under "Community Names," click Add, enter a community string (e.g., public), and set permissions (e.g., READ ONLY).
		b> Under "Accept SNMP packets from these hosts," click Add and enter the IP address of your Network Management System (NMS). 
		
3. Configure Windows Firewall
	Search for "Windows Defender Firewall with Advanced Security" and open it.
	Create a new Inbound Rule for Port (UDP).
	Specify ports 161, 162 and allow the connection for your NMS's IP address.
	
4.Using PowerShell (Alternative) ( run powershell as Adminstartor)
	Install: Add-WindowsCapability -Online -Name "SNMP.Client~~~~0.0.1.0".
	Verify: Get-WindowsCapability -Online | Where-Object Name -like "*SNMP*".
	Configure: Use services.msc as described above for security settings.
